title: Orray
description: A spatial platform for understanding and operating distributed systems.
icon: Map
---

Orray is a spatial interface for Kubernetes-based platforms. Instead of dashboards and tables, your system is a navigable canvas — zoom into services, trace requests visually, diff environments in 3D, and act on infrastructure directly.

<Callout type="info">
Orray is in early development. These docs capture research, design ideas, and technical decisions as the project takes shape.
</Callout>
